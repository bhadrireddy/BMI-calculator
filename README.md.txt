# BMI Calculator

A simple web-based **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**. This project allows users to calculate their BMI based on their weight (in kg) and height (in cm) and provides a category indicating whether they are underweight, normal, overweight, obese, severely obese, or morbidly obese.

---

## üìù Features
- Input **Name**, **Weight (kg)**, and **Height (cm)**
- Calculates **BMI** using the standard formula
- Displays **BMI value** and **weight category**
- Simple and clean user interface
- Fully responsive for desktop and mobile browsers

---

## üíª How to Run
1. Clone or download this repository.
2. Ensure the following files are in the same folder:
   - `bmi.html`
   - `style.css`
3. Open `bmi.html` in your web browser (Chrome, Firefox, Edge, etc.).
4. Enter your **name, weight, and height**, then click **Calculate BMI**.
